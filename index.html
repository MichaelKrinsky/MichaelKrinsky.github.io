Welcome To Michael Krinsky's Project Page

UCI Rocket Project

Since the start of Spring 2024, I've been the Chief Engineer for the UCI Rocket Project. Leading a team of 40 dedicated students, our mission is to break the undergraduate launch record for methalox rockets. Joining the team during my sophomore year, I have worked on a multitude of avionics tasks. Most notably, I developed the avionics system used in launching UCI's first ever liquid rocket where we achieved an altitude of 9300 feet.

UAV Forge Machine Learning

The UAV Forge team competes in the AUVSI aerial vehicle competition in which we work to design, manufacture, and program an autonomous UAV that uses machine learning to compete in aerial challenges. My role lies in developing the machine learning models and advanced vision processes that enables our UAV to detect, classify, and track payload dropzone identifiers accurately from the sky.

Sokoban AI

The Sokoban game was a machine learning program that I developed with Mugared Khalifa and Sam Perelgut for a project in Artificial Intelligence. The goal of the project was to design an AI that used reinforcement learning to learn how to play Sokoban, a notoriously difficult puzzle game.

The Grocery App

The Grocery App is an online grocery list you can use to easily keep track of items you need to buy at the store. Developed during the Coronavirus pandemic, the app allows several families or neighbors to share a list so they can shop for each other at the grocery store. I built it using Angular and Java Web Services.

The Banking Analysis App

The Banking Analysis App is a bank statement analyzer that automatically sorts your transactions into your own personalized categories. I developed this app to better keep track of my spending. I built it using Angular and Java Web Services.

Machine Learning Vision Project

The Machine Learning project was created to test an alternative form of vision processing using TensorFlow. I developed Pi Bot to track brightly colored balls, but he can be used to track virtually anything if you calibrate him correctly.
