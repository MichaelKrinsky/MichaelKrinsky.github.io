Pi Bot Project

On my robotics team, we used vision processing to help our robot navigate around the field and line itself up to a target. I created Pi Bot to learn more about how machine learning could be used as an alternative to vision processing. Pi Bot uses Tensor Flow machine learning to detect and trace objects and is controlled using a Raspberry Pi.


The machine learning software can be calibrated to detect all sorts of items like chairs and TV's, but Pi Bot uses it to detect sports balls.



While Pi Bot's intended purpose is to be be driven autonomously, he can be manually driven using a robot controller in the web browser for "emergency maneuvers". This custom made robot controller can even run on your phone and can access Pi Bot from anywhere with internet access.
